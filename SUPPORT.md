# Support

Kanera is source-available software. Community support is best-effort and depends on maintainer availability.

Use GitHub issues or discussions, when enabled, for:

- reproducible bugs;
- self-hosting questions;
- documentation problems;
- small feature requests.

Do not use public issues for security vulnerabilities. Follow [SECURITY.md](SECURITY.md) instead.

Commercial support, hosted-service support, private roadmap commitments, and custom licensing are handled separately by Happen Software Limited.
